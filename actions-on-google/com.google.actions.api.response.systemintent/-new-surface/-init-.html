NewSurface()

System intent response to request user to transfer to another surface with the specified capabilities. For example, the user may need to continue the conversation on a device with a screen output.

More details here.

Usage:

ResponseBuilder responseBuilder = getResponseBuilder();
responseBuilder.add(new NewSurface(
  Capability.SCREEN_OUTPUT.getValue(),
    "To show you an image",
    "Check out this image"));
